## Project Skills
* Vue-cli (檔案控管)
* Vue-router （進行頁面動態跳轉）
* Regular Expression （進行表單驗證）
  
## Project Features
### 首頁
* 進度條顯示現在進度
* 選擇付款方式（包含信用卡/超商/ATM）
### 填寫付款資訊
* 信用卡資訊（卡號/到期日期/確認碼/信箱）
* 銀行資訊（選擇銀行/信箱）
* 超商資訊（選擇超商/信箱）
* 沒有填寫完整不能送出表單
### 結果
* 顯示最終訂單資訊
